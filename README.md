# 🍬 Tier List Słodyczy do Biura

Agora na słodycze do biura! Głosowanie tygodniowe z drag-and-drop tier listą.

## Features
- 🗳️ 1 głos na osobę tygodniowo
- 🔒 Wymagane imię i nazwisko
- 🚫 Blokada duplikatów w ciągu tygodnia
- 📊 Podsumowanie wyników z średnimi
- 📱 Mobile-friendly drag & drop

## Deploy na Render.com

1. Wejdź na [render.com](https://render.com)
2. Kliknij "New Web Service"
3. Podłącz ten repo GitHub
4. Auto-deploy z `render.yaml`

Albo ręcznie:
- **Build Command:** `npm install`
- **Start Command:** `node server.js`

## Lokalnie

```bash
npm install
node server.js
# Otwórz http://localhost:3456
```
